TIPOS DE ETIQUETAS

En HTML y CSS tenemos diferente tipos de etiquetas cada uno con un propósito en particular una etiqueta se puede componerse de dos maneras.

ESTRUCTURA DE ETIQUETAS
Etiquetas que solo se apertura Etiquetas que tiene apertura y un cierre
Este tipo de etiqueta común mente se utilizan
para indicar una dirección u origen se
componen por argumento u opciones tales
como src href. 		Este tipo de etiquetas común mente se
utilizan para encerrar un contenido tales como
texto img entre otros.

Lista de etiquetas dirigidas al texto:

  • <p> </p> se utiliza para colocar un texto
  • <h1> </h1> se utiliza para colocar un titulo
  • <h2> </h2> se utiliza para colocar un titulo
  • <h3> </h3> se utiliza para colocar un titulo
  • <h4> </h4> se utiliza para colocar un titulo
  • <h5> </h5> se utiliza para colocar un titulo
  • <h6> </h6> se utiliza para colocar un titulo
  • <td> </td> se utiliza para poner texto en una tabla
  • <li> </li> se utiliza para enlistar cosas
  • <i> </i>: Muestra el texto marcado con un estilo en cursiva o itálica.
  • <b> </b>: Muestra el texto marcado con un estilo en negrita.
  • <u> </u>: Muestra el texto subrayado.
  • <mark> </mark> Representa un texto marcado o resaltado como referencia o anotación, debido a su relevancia o importancia.:
  • <body> </body> Es el cuerpo de la pagina en donde van casi todas las etiquetas

Listado de etiquetas dirigidas a objetos:

  • <img> se utiliza para poner una imagen
  • <header> </header> se utiliza para diferenciar de las diferentes secciones
  • <section> </section> se utiliza para hacer secciones
  • <head> </head>> delimita el encabezado del documento
  • <title> </title> título del documento (se muestra en la pestaña del navegador)
  • <table> </table> Sirve para crear una tabla
  • <ol> </ol> Sirve para crear una lista ordenada
  • <ul> </ul> Sirve para crear una lista desordenada

Listado de propiedades css dirigidas a texto:

  • color: Define el color de la letra
  • font-size: Define el tamaño de la letra
  • text-align: Define la posicion de la letra
  • font-family: Define la tipografia de la letra
  • text-decoration-line: Subrraya las palabras
  • Letter-spacing: Espacio entre caracteres normal
  • Word-spacing: Espacio entre palabras normal

Listado de propiedades dirigidas a objetos:

  • Background-color: Color del fondo de un objeto.
  • display: block;
  • Font-size: Define el tamaño de la fuente y el valor se puede escribir en pixels o en ems.
  • Width: Define el ancho de un elemento, el valor se puede escribir en pixels, ems o porcentaje
  • Height:Define el alto de un elemento, el valor se puede escribir en pixels, ems o porcentaje.
  • Padding:Es la distancia desde el borde de un elemento hasta su contenido.
  • Margin:Es la distancia entre un elemento y otro (desde el borde de un elemento hacia afuera)
  • Border:Define el borde de un elemento, su color, su estilo y grosor.
  • Background:Define los fondos de un objeto. El fondo puede ser una imagen o un color.

Listado de propiedades dirigidas a imagenes:

  • <img> Sirve para colocar imagenes en paginas web
  • <img src =""> Indica el nombre o la URL de la imagen a mostrar. Atributo obligatorio.
  • <img alt =""> Establece un texto alternativo que describa la imagen a mostrar. Atributo obligatorio.
  • <img width =""> Indica el ancho de la imagen. No se debe indicar unidad. Se aconseja hacerlo desde CSS.
  • <img height =""> Indica el alto de la imagen. No se debe indicar unidad. Se aconseja hacerlo desde CSS.
